# 06 — Grille qualité épisode (Agent : Responsable qualité)

_À remplir après tournage, AVANT publication. Note /100. Seuil de publication : 80._

---

## Grille de notation (/100)

| # | Critère | Poids | Note | Ce qu'on regarde |
|---|---------|------:|:----:|------------------|
| 1 | **Valeur du récit** | /20 | | Y a-t-il une vraie histoire, un arc, une progression ? |
| 2 | **Intensité émotionnelle** | /20 | | Au moins un moment de vérité / d'émotion authentique ? |
| 3 | **Valeur éducative** | /15 | | Le spectateur apprend-il quelque chose d'actionnable ? |
| 4 | **Inspiration** | /15 | | Ressort-on plus motivé, plus lucide ? |
| 5 | **Potentiel viral** | /10 | | Existe-t-il des séquences « clippables » fortes ? |
| 6 | **Qualité journalistique** | /10 | | Faits justes, équilibre, honnêteté, respect ? |
| 7 | **Pertinence culturelle** (lien doc 07) | /5 | | Résonne avec le Mali / l'Afrique ? Pas de faux pas ? |
| 8 | **Exécution & rythme** | /5 | | Montage, longueurs, lisibilité, signature de marque ? |
| | **TOTAL** | **/100** | | |

## Décision

| Note | Décision |
|------|----------|
| **90–100** | 🌟 Publier en épisode phare + push marketing fort. |
| **80–89** | ✅ Publier. |
| **70–79** | 🟠 Publier seulement après corrections de montage ciblées. |
| **< 70** | ❌ Ne pas publier en l'état. Décider : remontage profond, segment seul, ou archive. |

> ⚠️ **Veto qualité** : ne jamais publier sous 80/100. Mieux vaut décaler un
> épisode que diluer la marque. La régularité se gère par la banque d'épisodes
> d'avance (doc 00), jamais en sacrifiant la barre de qualité.
