# 02 — Grille d'évaluation des invités (Agent : Directeur éditorial)

_Outil de décision. À remplir pour chaque invité envisagé. Note sur 100._

---

## Mode d'emploi
1. Noter chaque critère selon son poids.
2. Additionner → note /100.
3. Appliquer le seuil de décision.
4. **Justifier systématiquement** la décision en 3–5 lignes.

## Grille de notation (/100)

| # | Critère | Poids | Note (0 → max) | Question de contrôle |
|---|---------|------:|:--------------:|----------------------|
| 1 | **Histoire exceptionnelle** | /20 | | Le parcours sort-il vraiment de l'ordinaire ? Y a-t-il un récit ? |
| 2 | **Impact réel & vérifiable** | /15 | | A-t-il/elle changé quelque chose de concret et mesurable ? |
| 3 | **Crédibilité** | /15 | | Les faits tiennent-ils ? Réputation solide ? Risque de mensonge faible ? |
| 4 | **Capacité à inspirer** | /15 | | Le spectateur ressort-il grandi, motivé ? |
| 5 | **Originalité** | /10 | | Apporte-t-il un angle qu'on n'entend jamais ailleurs ? |
| 6 | **Intérêt pour le public malien** | /10 | | Résonne-t-il avec le vécu et les références du Mali ? |
| 7 | **Potentiel panafricain / diaspora** | /10 | | L'histoire dépasse-t-elle les frontières ? |
| 8 | **Capacité d'expression & présence** | /5 | | Sait-il/elle raconter ? À l'aise face caméra ? |
| | **TOTAL** | **/100** | | |

## Seuils de décision

| Note | Décision | Action |
|------|----------|--------|
| **85–100** | ✅ **VALIDER — prioritaire** | Programmer en tête de saison. Invité « locomotive ». |
| **75–84** | ✅ **VALIDER** | Programmer. Bon épisode attendu. |
| **60–74** | 🟠 **À RETRAVAILLER** | Bon potentiel mais angle à resserrer / preuves à consolider / moment à trouver. Re-noter après. |
| **< 60** | ❌ **REFUSER** | Ne correspond pas à la ligne aujourd'hui. Garder en veille si le parcours évolue. |

> ⚠️ **Veto qualitatif** : quelle que soit la note, refuser si l'invité présente
> un risque éthique majeur non maîtrisable, une incompatibilité avec la charte de
> neutralité (doc 01/07), ou si la véracité des faits clés ne peut être établie.

## Exemple type (illustratif)
> **Invité fictif** : une femme ayant bâti la première PME agro-alimentaire
> exportatrice depuis un village, partie de rien, employant 200 personnes.
> - Histoire exceptionnelle : 18/20 · Impact : 14/15 · Crédibilité : 13/15 ·
>   Inspirer : 14/15 · Originalité : 8/10 · Public malien : 10/10 ·
>   Panafricain : 7/10 · Présence : 4/5 → **TOTAL 88/100 → VALIDER prioritaire.**
> - _Justif. :_ récit ascensionnel rare, impact local massif et vérifiable,
>   modèle réplicable dans toute la sous-région, forte charge émotionnelle.
> - _Angle :_ « Comment on bâtit un empire depuis un endroit où personne ne
>   croyait que c'était possible. »
